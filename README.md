# Intelligent Resume Screening and Candidate Matching System

## Overview
This project is an AI-based backend system that automates resume screening and evaluates candidate suitability against job descriptions.

## Features
- Resume PDF parsing
- Skill extraction
- Candidate-job matching score
- RESTful API using FastAPI
- SQLite database integration

## Tech Stack
- Python
- FastAPI
- NLP techniques
- SQLite

## How to Run

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
